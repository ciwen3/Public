# RUST_BACKTRACE
## To Enable BACKTRACE:
type into cmd: 
```bash
'set RUST_BACKTRACE=1'
```
confirm with [enter]
then type 
```bash
'cargo run'
```

## To Disable BACKTRACE:
type into cmd: 
```bash
'set RUST_BACKTRACE=0'
```
now 'cargo run' will output 'original' error messages.
